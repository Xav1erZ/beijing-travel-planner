# 从头开始：GitHub Pages 部署（超简单版）

## 第1步：检查你的文件
打开文件夹：`C:\Users\10403\CodeBuddy\20260402163929`
你应该看到这些文件：
- ✅ `index.html`（最重要的文件）
- ✅ `style.css`
- ✅ `script.js`
- ✅ 其他文件

## 第2步：打开GitHub Desktop
1. 在Windows搜索栏输入 `GitHub Desktop`
2. 打开它

## 第3步：在GitHub Desktop中添加仓库
1. 点击左上角 **File**（文件）
2. 选择 **Add Local Repository**（添加本地仓库）
3. 点击 **Choose**（选择）
4. 找到并选择：`C:\Users\10403\CodeBuddy\20260402163929`
5. 点击 **Add Repository**（添加仓库）

## 第4步：提交文件
在GitHub Desktop中：
1. 左边会显示所有文件（9个文件）
2. 在左下角 **Summary** 框中输入：
   ```
   我的AI北京三日游网站
   ```
3. 点击 **Commit to main**（提交到主分支）

## 第5步：推送到GitHub
1. 点击右上角 **Push origin**（推送到源）
2. 等待完成（几秒钟）

## 第6步：打开浏览器
1. 打开Chrome或Edge浏览器
2. 访问：[https://github.com/Xav1erZ/beijing-travel-planner](https://github.com/Xav1erZ/beijing-travel-planner)

## 第7步：配置GitHub Pages
1. 在仓库页面，点击 **Settings**（设置）
2. 左边菜单点击 **Pages**
3. 设置：
   - **Source**：选择 `Deploy from a branch`
   - **Branch**：选择 `main`
   - **Folder**：选择 `/ (root)`
4. 点击 **Save**（保存）

## 第8步：等待
等待 **2分钟**（去喝口水）

## 第9步：访问你的网站
在浏览器中输入：
```
https://xav1erz.github.io/beijing-travel-planner/
```

## 第10步：检查是否成功
你应该看到：
- 蓝色标题 "AI北京三日游策划"
- 导航菜单
- "生成AI行程计划"按钮
